# ACppMakeTemplate

## Getting Started 


```
git clone https://github.com/HajerZhang/ACppMakeTemplate.git
```

### Prerequisites 

```
GNU make
```

### How to use

Windows  & Linux:

```sh
make
```
Debug mode:

```sh
make debug
```
How to run:

```sh
./main.exe
```
Clean:

```sh
make clean
```
